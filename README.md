# Sales-Data-Analysis
# Project Overview

This project demonstrates basic data analysis using Python and the Pandas library. A sales dataset is loaded from a CSV file, cleaned, explored, and analyzed to generate useful business insights such as total revenue, best-selling products, and average sales.

# Objectives
Learn data analysis using Pandas
Load CSV files
Explore dataset information
Handle missing values
Calculate sales metrics
Generate a simple business report
